# Theories of Human Development

## Theory vs. Hypothesis

- **Theory** = broad, organized explanation of many phenomena.
- **Hypothesis** = narrower prediction; can be disproven.

------

# Psychodynamic Theories

- Emphasize **past experiences + emotions** shaping behavior.
- Include **psychoanalytic** theories.

## Sigmund Freud — Psychoanalytic Theory

- Personality shaped mainly in **first 6 years**.
- Behavior driven by **unconscious motives/drives**.

### Freud’s Psychosexual Stages

1. **Oral (0–1)** → pleasure from mouth/sucking.
2. **Anal (1–3)** → toilet training/anus.
3. **Phallic (3–6)** → focus on genitals.
4. **Latency (6–11)** → sexual urges quiet.
5. **Genital (adolescence+)** → mature sexuality.

### Key Idea

- Conflicts at each stage shape adult personality.
- Examples:
  - weaning → oral conflict
  - toilet training → anal conflict

### Freud Today

Accepted:

- **repression**
- **rationalization**

Criticized:

- psychosexual ideas
- belief development stops after puberty

------

## Erik Erikson — Psychosocial Theory

- Expanded Freud’s ideas.
- Focused on **social + cultural influences**, not sexuality.
- Development continues across lifespan.

### Main Idea

- Each stage has a conflict (“vs.”).
- Outcomes are usually between extremes.

### Erikson’s Stages

| Age          | Conflict                    |
| ------------ | --------------------------- |
| Infancy      | Trust vs. Mistrust          |
| Toddler      | Autonomy vs. Shame/Doubt    |
| Preschool    | Initiative vs. Guilt        |
| School Age   | Industry vs. Inferiority    |
| Adolescence  | Identity vs. Role Confusion |
| Young Adult  | Intimacy vs. Isolation      |
| Middle Adult | Generativity vs. Stagnation |
| Older Adult  | Integrity vs. Despair       |

### Example

- Trouble with intimacy in adulthood may connect to unresolved trust issues in infancy.

# Behaviorism / Learning Theory

- Opposes Freud’s focus on unconscious urges.
- Emphasizes **learning through environment + responses from others**.
- People can learn at any age.

## Types of Learning

| Type                   | Main Idea                                 |
| ---------------------- | ----------------------------------------- |
| Classical Conditioning | Learning by association                   |
| Operant Conditioning   | Learning through reinforcement/punishment |
| Social Learning        | Learning by observing others              |

------

## Ivan Pavlov — Classical Conditioning

- Dogs learned to associate footsteps/sounds with food.
- Neutral stimulus → conditioned response.

### Example

- Bell → salivation after repeated pairing with food.

------

## Operant Conditioning

- Behavior shaped by consequences.

### Key Terms

- **Reinforcement** = increases behavior.
- **Punishment** = decreases behavior.

### Important Point

- A consequence is only punishment if behavior decreases.
- What works differs by person.

------

## Albert Bandura — Social Learning Theory

- People learn by **observing models**.
- Reinforcement not always needed.

### Example

- Children exposed to violence may imitate, resist, or avoid it.

------

# Cognitive Theory

## Jean Piaget

- Children actively try to understand the world.
- Thinking changes with **maturation + experience**.

### Core Idea

- How people think affects behavior.

------

## Piaget’s Cognitive Stages

| Age  | Stage                | Key Feature                   |
| ---- | -------------------- | ----------------------------- |
| 0–2  | Sensorimotor         | object permanence             |
| 2–6  | Preoperational       | symbolic thought, egocentrism |
| 6–11 | Concrete Operational | logical thinking              |
| 12+  | Formal Operational   | abstract reasoning            |

------

## Cognitive Processes

### Assimilation

- Fit new experiences into existing ideas.
- Maintains **equilibrium** (mental balance).

### Accommodation

- Adjust thinking for new information.
- Happens during **disequilibrium**.

### Example

- New experiences challenge old beliefs → cognitive growth.

------

# Evolutionary Theory

## Charles Darwin

- Humans evolved behaviors that support **survival + reproduction**.

### Example

- Humans fear snakes more naturally than cars because fear responses evolved long ago.
